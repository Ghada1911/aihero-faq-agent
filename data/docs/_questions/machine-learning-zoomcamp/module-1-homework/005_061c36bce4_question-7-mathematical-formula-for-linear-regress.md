In Question 7, we are asked to calculate the following:

- `X^T X`
- `(X^T X)^{-1}`
- `w = (X^T X)^{-1} X^T y`

The initial problem `w = X^{-1} y` can be solved by this, where a matrix `X` is multiplied by some unknown weights `w` resulting in the target `y`.
